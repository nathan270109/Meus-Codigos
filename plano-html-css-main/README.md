# plano-html-css

## GitHub Pages

O **GitHub Pages** é gratuito, permite **personalizar o domínio** e é **amplamente usado** por estudantes e profissionais de desenvolvimento. Adamsilver

### Passo a passo

1. **Crie uma conta** em https://github.com  
2. **Faça um clone do GitHub**: https://karizeviecelli.github.io/plano-html-css/  
3. **Salve em sua máquina**.  
4. Clique em **New repository** → nome: **plano-html-css**  
5. Marque **Add a README file** → clique em **Create repository**  
6. Clique em **uploading an existing file** → arraste o **index.html** → clique em **Commit changes**  
7. Vá em **Settings** → **Pages** → **Branch: main** → **Save**  

Em **~1 minuto** o site estará em:  
**https://seu-usuario.github.io/plano-html-css**

**Vantagem:** link permanente, profissional, e você pode atualizar o arquivo quando quiser.
